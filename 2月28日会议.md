### API Demo

| 方法名          | 意义                                                            | 参数                   | 返回         |
| ------------ | ------------------------------------------------------------- | -------------------- | ---------- |
| getblockhash | Returns hash of block in best-block-chain at height provided. | height-int, required | hex-string |



### RPC-Client

1. 定义getblockhash请求类型

2. 根据不同api封装请求并发送



### RPC Server

1. 接受请求，解析请求的API名称，参数值

2. **调用**其它模块的函数

3. 返回结果和error
